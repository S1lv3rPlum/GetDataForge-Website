The Juke

The Juke is a 2-part App for Live music venues.

For Bands:

  • The Juke enables you to develop a set list (with breaks listed) so listeners can get excited for what's coming up
  • Set which songs the audience members can request
  • Collect tips digitally

For Audience:

  • The Juke enables you to see the band's set list so you can plan your time on the dance floor
  • Request songs without leaving your table
