What is the DataForge Game Agent?

The DataForge Game Agent is an AI that learns to play games through trial and error — the same fundamental approach used by research teams at DeepMind and OpenAI. Instead of being programmed with rules, it figures them out on its own by playing thousands of games and learning from the results.

This is a working implementation of Deep Q-Learning (DQN) — the algorithm behind some of the most impressive AI achievements of the last decade. Built from scratch as a learning and demonstration project, with full source code available on GitHub.

Features

🎮

Game Launcher

Clean visual launcher to pick your game, difficulty, and agent. One command starts everything.

🧠

Two Agents

Random Agent as a baseline. DQN Learning Agent that genuinely improves over time.

📊

Live Dashboard

Real time graphs — win rate, reward, steps, correct flags. Compare both agents on one chart.

💣

Minesweeper

Three difficulty levels. Agent reveals cells and flags mines with a full reward and penalty system.

⚡

GPU Accelerated

Automatically uses NVIDIA GPU if available. Falls back to CPU seamlessly.

💾

Saves Progress

Model saves every 100 episodes per difficulty level. Resume training anytime.

How to Use It

  1. Download DataForge-Game-Agent.exe from the button above
  2. Double click the exe — no installation required
  3. The launcher opens — click Minesweeper
  4. Choose your agent — Random, Learning, or Both
  5. Select a difficulty and watch it play

Start with the Random Agent to establish a baseline, then switch to the Learning Agent and watch it improve over hundreds of games. Running Both simultaneously shows both agents on a shared dashboard for direct comparison.

System Requirements

Windows 10 or 11 • 4GB RAM minimum, 8GB recommended • Any modern processor • NVIDIA GPU optional but recommended for faster training • Approximately 200MB disk space

Open Source

The full source code is available on GitHub, built with Python, PyTorch, Pygame, and Gymnasium. The project is actively developed with new games and features planned.
