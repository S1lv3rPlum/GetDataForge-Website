What is the DataForge Game Agent?

The DataForge Game Agent is an AI that learns to play games through trial and error — the same approach used by research teams at DeepMind and OpenAI to build world-class game-playing AI. Instead of being programmed with rules, it figures them out on its own by playing thousands of games and learning from the results.

This is a real, working implementation of Deep Q-Learning (DQN) — the algorithm that made headlines when it taught itself to play Atari games better than humans. Built from scratch as a learning project, with full source code available.

Features

🎮

Game Launcher

Clean selection screen to pick your game, difficulty, and agent type. One command launches everything.

🧠

Two Agents

Random Agent clicks randomly as a baseline. DQN Learning Agent actually learns and improves over time.

📊

Live Dashboard

Real time graphs showing win rate, reward, steps, and correct flags. Watch both agents on the same chart.

💣

Minesweeper

Three difficulty levels — Beginner, Medium, Hard. Agent can reveal cells and flag mines with correct/incorrect rewards.

⚡

GPU Accelerated

Automatically uses your NVIDIA GPU if available for faster training. Falls back to CPU seamlessly.

💾

Saves Progress

Model saves every 100 episodes per difficulty. Resume training anytime — progress is never lost.

How to Use It

  1. Download the DataForge-Game-Agent.exe file above
  2. Double click the exe — no installation needed
  3. The launcher opens — pick Minesweeper
  4. Choose your agent — Random, Learning, or Both
  5. Select a difficulty and watch it play!

The Random Agent is a good starting point — watch it flail around randomly so you have a baseline to compare against. Then switch to the Learning Agent and watch it gradually improve over hundreds of games.

System Requirements

Windows 10 or 11 • 4GB RAM minimum (8GB recommended) • Any modern processor • NVIDIA GPU optional but recommended for faster training • ~200MB disk space

Open Source

The full source code is available on GitHub. Built with Python, PyTorch, Pygame, and Gymnasium. Contributions and feedback welcome!
