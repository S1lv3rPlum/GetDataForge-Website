Child Safety

🛡️ How We Keep Kids Safe

🔒 Content Filtering 👨‍👩‍👧 Parent Monitoring 🚫 Word Blocking 📊 Search History ✅ COPPA Compliant

Triple-Layer Filtering: SafeSearch API + keyword filtering + image analysis working together.

Curated Sources: Only searches trusted, family-friendly image databases like Pixabay.

No Chat or Social: Pure search only — no communication features of any kind.

Parent Dashboard: PIN-protected area to view search history and manage blocked words.

No Data Collection: We don't track, store, or sell your child's information. Search history stays on your device only.
