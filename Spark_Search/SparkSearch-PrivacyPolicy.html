Privacy Policy

Last Updated: January 4, 2026

Simple Summary: Spark Search is designed with your child's privacy as our top priority. We do not collect, store, or share any personal information. All data stays on your device, and parents have full control over what their children can access.

1. Introduction

Welcome to Spark Search, a safe search application for children created by DataForge Apps. This Privacy Policy explains how we handle information when you and your child use our application.

Spark Search is designed to comply with the Children's Online Privacy Protection Act (COPPA) and prioritizes the safety and privacy of children using our service.

2. Information We Do NOT Collect

Spark Search is built on a privacy-first foundation. We explicitly do not collect, store, or transmit:

  • Names, email addresses, or any personal identification
  • User accounts or login credentials
  • Location data or GPS information
  • Device identifiers or advertising IDs
  • Photos, videos, or any media from your device
  • Contact lists or phone numbers
  • Browsing history outside of local storage
  • Any information that could identify your child

3. Local Data Storage

Spark Search stores the following data locally on your device only:

Search History

  • Search terms your child has entered
  • Timestamps of searches
  • Content type searched (pictures, coloring pages, videos, or songs)
  • This data is stored for up to 90 days and automatically deleted thereafter
  • Parents can view, export, or delete this history at any time through Parent Settings

Parent Settings

  • Parent PIN (stored encrypted on device)
  • Blocked word list (words parents have chosen to restrict)
  • Theme preferences
  • Content filter sensitivity settings

Important Notes About Local Storage:

  • All data remains on your device and is never transmitted to our servers
  • Uninstalling the app will permanently delete all stored data
  • We cannot access, view, or retrieve any data stored on your device
  • Data is not backed up to cloud services

4. Future Features: Optional Cloud Sync

We are developing an optional premium feature called "Family Sync" that will allow parents to:

  • Sync search history across multiple devices
  • Monitor searches from any device in the family
  • Sync blocked word lists across devices
  • Receive weekly email summaries of search activity

This feature will be:

  • Completely optional (opt-in only)
  • Require explicit parent consent
  • Only accessible with parent authentication
  • Clearly labeled and explained before enabling
  • Deletable at any time by the parent

We will update this Privacy Policy before launching Family Sync, and existing users will be notified of changes.

5. How We Keep Content Safe

Spark Search uses multiple layers of protection to ensure age-appropriate content:

Content Sources

  • We search only curated, family-friendly image databases (currently Pixabay)
  • All API calls use strict SafeSearch parameters
  • Content providers are vetted for child safety

Content Filtering

  • Keyword filtering: Automatically blocks searches and results containing inappropriate terms
  • Image analysis: Content is screened before display
  • Manual blocking: Parents can add custom blocked words
  • Ongoing monitoring: We continuously update our safety filters

6. Third-Party Services

Spark Search uses the following third-party services to provide functionality:

Image Search: Pixabay API

  • Used to search for images and coloring pages
  • SafeSearch enabled on all requests
  • Does not receive any personal information about users
  • Only receives search terms entered by users
  • Privacy Policy: pixabay.com/service/privacy/

Future Services (Coming Soon)

  • YouTube Data API: For safe video searches (with strict SafeSearch)
  • Google Cloud Vision API: For enhanced image content analysis

When these services are added, we will update this policy with details about their privacy practices.

7. No Advertising or Tracking

Spark Search does not:

  • Display advertisements of any kind
  • Use analytics or tracking tools
  • Share data with advertisers or marketing companies
  • Build user profiles or behavioral data
  • Use cookies (web version may use essential session cookies only)

8. Parent Rights and Controls

As a parent or guardian, you have complete control over your child's use of Spark Search:

Access Parent Settings

  • Set a PIN to protect parent-only features
  • View complete search history
  • Export search history data
  • Delete individual searches or entire history

Content Controls

  • Add words or topics to block list
  • Remove words from block list
  • Adjust content filter sensitivity
  • Enable or disable specific content types (pictures, coloring pages, etc.)

Data Management

  • Clear all stored data at any time
  • Uninstall app to permanently delete all data
  • No data recovery possible after deletion

9. Data Security

We take security seriously:

  • Parent PINs are stored using encryption
  • Local data is protected by device-level security
  • App does not have internet access beyond necessary API calls
  • No data is transmitted without explicit user action
  • All API communications use secure HTTPS connections

10. Age Restrictions

Spark Search is designed for children ages 3-12 years old. We recommend:

  • Parents set up the app and Parent Settings before allowing use
  • Parents review blocked words and content filters
  • Young children (3-6) use the app with parent supervision
  • Parents periodically review search history

11. Changes to This Policy

We may update this Privacy Policy as we add new features or improve the app. When we make changes:

  • We will update the "Last Updated" date at the top
  • Significant changes will be announced in the app
  • Parents will be notified before any data collection practices change
  • Continued use after changes constitutes acceptance

12. Compliance

Spark Search is designed to comply with:

  • COPPA (Children's Online Privacy Protection Act)
  • GDPR (General Data Protection Regulation) - for EU users
  • Google Play Family Policies
  • Apple App Store Guidelines (when iOS version launches)

13. Beta Testing

Spark Search is currently in beta testing. During this phase:

  • The app may contain bugs or incomplete features
  • We may collect anonymous crash reports to improve stability (opt-in only)
  • Beta testers may be asked for feedback via email (no child data shared)
  • All privacy protections remain in effect
